[Community HCL](community-hcl)
The main landing page for the community-maintained Hardware Compatibility List, providing links to supported systems and components.

[Getting Hardware Info](getting-hardware-info)
Guide on using the Device Driver Utility (DDU) to identify your hardware and its compatibility status with OpenIndiana drivers.

[Components](components)
A detailed list of individual hardware parts, such as network cards, storage controllers, and graphics adapters, confirmed to work by the community. ** May be outdated.**
[Systems](systems)
A categorized list of complete computer systems (Laptops, Workstations, and Servers) that have been tested and verified by users.
